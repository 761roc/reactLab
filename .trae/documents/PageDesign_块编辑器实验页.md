## 全局设计规范（适用于本功能页）

### Layout
- 桌面端优先：内容区最大宽度 1200–1400px，左右分栏为主。
- 布局建议：CSS Grid（两列）+ Flex（局部对齐）。
- 间距：8px 基础栅格；卡片/面板内边距 16–20px；分栏间距 16–24px。

### Meta Information
- Title：块编辑器实验（类 Notion/飞书）
- Description：块级编辑、预设块、选中样式修改与文档导入导出实验页
- Open Graph：使用同 Title/Description；可复用站点默认 OG 图

### Global Styles
- 背景：#0B0F19（深色）或项目默认背景；内容卡片：#111827 / #FFFFFF（按全局主题）
- 字体：系统字体栈；正文 14–16px；标题 18–24px
- 主色：#3B82F6；强调/高亮：#F59E0B；危险：#EF4444
- 按钮：默认/主按钮/危险三态；hover 提升亮度或增加阴影
- 链接：主色下划线 hover
- 过渡：hover/选中态 120–180ms ease-out

## Page 2：块编辑器实验页

### Layout
- 顶部：页面头部（标题 + 导入/导出操作区）
- 主体两列（Grid）：
  - 左列（主列）：编辑区（Block Canvas）
  - 右列（侧栏）：预设功能块 + 选中样式修改（可分为两个卡片区块）

### Page Structure
1. Header（固定在内容顶部，不需要吸顶）
2. Main Grid（左编辑右工具）
3. 提示/错误区（建议用 Toast 或页面内 Alert）

### Sections & Components

#### 1) Header：标题与导入导出
- 标题：块编辑器实验（类 Notion/飞书）
- 说明文本：一句话描述本页用途
- 操作按钮组（右侧对齐）：
  - 导入：按钮 + 隐藏 file input（接受 .json/.md 等）
  - 导出：下拉（JSON/Markdown）或单按钮（先做一种）
  - 可选：清空文档（危险按钮，带二次确认）

#### 2) 左列：编辑区（Block Canvas）
- 容器：卡片/纸张风格（内边距 20–24px），支持纵向滚动
- Block 渲染：
  - 块之间 8–12px 间距
  - hover：块外框浅色描边
  - 选中块：外框高亮（与 selection 高亮区分）
- 编辑交互：
  - 光标所在块显示轻量占位提示
  - 回车创建新块；退格在空块时合并到上一个块（按编辑器能力实现）

#### 3) 右列上半：预设功能块
- 卡片标题：预设功能块（用于快速插入）
- 块列表（按钮/列表项）：标题、段落、待办、引用/Callout、分割线
- 插入规则：
  - 点击后插入到当前光标位置（若无光标则追加到末尾）
  - 插入后将焦点移动到新块可编辑区域

#### 4) 右列下半：选中样式修改
- 卡片标题：选中样式（选中后可调整）
- 空状态：未选中时提示“选择文本或块以编辑样式”
- 文本样式控件（按实现优先级）：
  - Toggle：加粗/斜体/下划线/行内代码
  - 颜色：文字颜色/背景高亮（简化为少量色板）
- 块级样式（可选，若实现）：
  - 对齐方式/缩进/块类型切换（与预设块联动时需避免冲突）

#### 5) 反馈与错误
- 导入失败：显示错误原因（格式不合法/版本不兼容等）
- 导出成功：Toast 提示或按钮状态反馈
